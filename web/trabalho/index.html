Curso Fullstack

O curso foi modelado especialmente para pessoas que tem o interesse em ingressar na área de desenvolvimento Web e Mobile integrando seu projeto de software com uma das tecnologias mais utilizadas para os recursos de “Back-End” o “Spring Boot”. Durante o curso o aluno ira desenvolver um sistema de compra e venda de mercadorias, gerenciando seu financeiro e estoque. Este projeto conta com 4 módulos para cada tecnologia especifica que será utilizada neste projeto, são eles:

Java OOP

Angular

Spring Boot

Android Studio
